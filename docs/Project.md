# GeoScan Gold - Описание проекта

## Обзор

GeoScan Gold - система геологического анализа и прогнозирования перспективных зон для поиска полезных ископаемых.

## Архитектура

*Будет заполнено по мере разработки*

## Технологический стек

- Python 3.10+
- NumPy, Pandas - обработка данных
- scikit-learn - машинное обучение
- Ruff - линтинг и форматирование
- MyPy - проверка типов
- Pytest - тестирование

## Статус разработки

Проект в начальной стадии разработки.
