# 🚀 FileZilla FTP Upload — Быстрая Инструкция

**Время:** 30-45 мин (вместо 1-2 часов через браузер)

---

## Шаг 1: Скачать FileZilla

**URL:** https://filezilla-project.org/

1. Click **"Download FileZilla Client"**
2. Запустить установщик
3. Install (все настройки по умолчанию)
4. Запустить FileZilla

---

## Шаг 2: Подключиться к Galaxy FTP

**В FileZilla (верхняя панель):**

```
Host:     ftp.usegalaxy.org
Username: твой email (из Galaxy)
Password: твой пароль
Port:     21
```

**Click:** "Quickconnect" (быстрое соединение)

---

## Шаг 3: Найти Файлы

**Левая панель (локальные файлы):**
- Перейти в: `D:\ДНК\fastq_data\raw\`

**Правая панель (Galaxy FTP):**
- Оставить: `/` (root)

---

## Шаг 4: Загрузить Файлы

**Перетащить слева направо:**

```
SRR12935486_1.fastq.gz  →  /
SRR12935486_2.fastq.gz  →  /
SRR12935488_1.fastq.gz  →  /
SRR12935488_2.fastq.gz  →  /
SRR12935490_1.fastq.gz  →  /
SRR12935490_2.fastq.gz  →  /
```

**Прогресс:**
- Внизу: вкладка "Transferring files"
- Зелёные галочки = готово

---

## Шаг 5: Проверить на Galaxy

1. Открыть https://usegalaxy.org/
2. Login
3. Посмотреть **History** (справа)
4. Файлы должны появиться через 5-10 мин после загрузки

---

## ⚡ Советы

**Ускорение:**
- Загружать все 6 файлов **одновременно** (выделить все → перетащить)
- FileZilla сам оптимизирует

**Если обрывается:**
- FileZilla автоматически возобновляет
- Можно вручную: Right-click → "Resume"
