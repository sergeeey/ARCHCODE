# RS-05: Topology Analyzer

**Status:** Open  
**Priority:** Medium  
**Risk ID:** S1

## Goal

Анализатор топологии: детекция TAD, сравнение с экспериментальными данными.

## Background

TAD detection:
- Из Hi-C данных
- Из симуляции экструзии
- Сравнение паттернов

## Engineering Unknowns

### S1: TAD Boundary Determinism
- Почему границы стабильны/вариабельны?
- Факторы стабильности?

## Method

### 1. TAD Detection

Алгоритм детекции:
- Insulation score
- Boundary calling
- TAD assignment

### 2. Comparison Engine

Сравнение:
- Simulated vs Experimental TADs
- Boundary positions
- Contact patterns

### 3. Validation Metrics

Метрики:
- Boundary overlap
- TAD size distribution
- Contact map correlation

## Deliverables

- `bin/topology_analyzer` - TAD analyzer (target C++)
- `src/archcode_core/tad_detector.py` - Python implementation
- Comparison tools
- Validation metrics

## Integration

- Input: Hi-C data, extrusion events
- Output: TAD boundaries, comparison metrics
- Used by: Model validation, RS-01

## Success Criteria

- Detects TADs from Hi-C
- Compares simulated vs experimental
- Validates model accuracy
