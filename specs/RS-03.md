# RS-03: Non-B DNA Barrier Model

**Status:** Open  
**Priority:** High  
**Risk IDs:** S3, L1

## Goal

Модель формирования барьеров из non-B DNA структур (G4, Z-DNA, R-loops).

## Background

Non-B DNA структуры формируют барьеры:
- G4: G-quadruplexes
- Z-DNA: Left-handed DNA
- R-loops: RNA-DNA hybrids

## Method

### 1. G4 Detection

Определить G4:
- Sequence motifs (GGG, GGGG)
- Минимальная длина
- Вероятность формирования

### 2. Z-DNA Detection

Определить Z-DNA:
- Sequence (GCGCGC)
- Negative supercoiling
- Transcription boost
- Ionic conditions

### 3. R-loop Detection

Определить R-loops:
- Transcription-dependent
- G-rich sequences
- Barrier strength

### 4. Barrier Hierarchy

Установить иерархию:
- G4 > Z-DNA > R-loops
- Конфликты разрешаются по приоритету
