# RS-02: TE Grammar Dictionary Construction

**Status:** Open  
**Priority:** High  
**Risk ID:** S1

## Goal

Построить полный словарь TE-мотивов для TE Grammar Engine.

## Background

TE Grammar Engine требует полного словаря транспозонных мотивов для анализа некодирующего генома. Текущая реализация использует placeholder мотивы.

## Method

1. **Data Collection**
   - Integrate RepeatMasker annotations
   - Integrate Dfam database
   - Collect species-specific TE libraries

2. **Motif Extraction**
   - Extract consensus sequences for each TE family
   - Build motif patterns (regex/sequence)
   - Assign strength scores based on conservation

3. **Dictionary Construction**
   - Build TE motif registry
   - Create hierarchical classification (LTR, LINE, SINE, etc.)
   - Map motifs to boundary effects

## Deliverables

- `te_grammar/motif_dictionary.py` - Main motif dictionary
- `te_grammar/motif_registry.yaml` - YAML registry of motifs
- `te_grammar/te_classifier.py` - TE classification module
- `tests/test_motif_dictionary.py` - Unit tests

## Constraints

- Модульная архитектура
- YAML-driven configuration
- Expandable registry (easy to add new motifs)
- Production-quality code

## Dependencies

- RepeatMasker output files
- Dfam database
- Genomic sequence data

## Success Criteria

- Dictionary covers >80% of annotated TEs
- Registry is easily expandable
- Integration with TE Grammar Engine
- Full test coverage ≥60%
