# RS-03: Non-B DNA Energy Barrier Model

**Status:** Open  
**Priority:** Medium  
**Risk ID:** L1, L2

## Goal

Реализовать энергетические барьеры для G4, Z-DNA и R-loops с калибровкой порогов.

## Background

Non-B DNA структуры создают энергетические барьеры для loop extrusion. Необходимо моделировать их формирование и влияние на границы TAD.

## Method

1. **G4 Modeling**
   - Implement G4 formation energy calculation
   - Parameterize stability threshold
   - Add sequence motif detection

2. **Z-DNA Modeling**
   - Implement B → Z-DNA transition threshold
   - Model torque-dependent transition
   - Add GC-rich motif detection

3. **R-loop Modeling**
   - Implement R-loop formation probability
   - Model G-rich sequence detection
   - Add barrier strength calculation

4. **Integration**
   - Combine barriers into hierarchy
   - Implement additive interaction model
   - Integrate with archcode_core

## Deliverables

- `nonB_logic/g4_model.py` - G4 formation model
- `nonB_logic/zdna_model.py` - Z-DNA transition model
- `nonB_logic/rloop_model.py` - R-loop formation model
- `nonB_logic/barrier_hierarchy.py` - Barrier interaction logic
- `config/nonB_logic.yaml` - Calibrated parameters

## Constraints

- Все параметры в YAML
- Строгие интерфейсы
- Energy-based calculations (not just sequence matching)
- Production-quality code

## Dependencies

- Literature values for energy barriers
- Sequence motif databases
- Calibration datasets

## Success Criteria

- Models predict barrier locations with >60% accuracy
- Parameters calibrated from literature
- Clean integration with archcode_core
- Full test coverage ≥60%
