# RS-01: Cohesin Loader Determinism

**Status:** Open  
**Priority:** High  
**Risk ID:** P1

## Goal

Вычислить правила выбора NIPBL-сайтов загрузки когезина для моделирования loop extrusion.

## Background

Loop extrusion требует знания мест загрузки когезина (NIPBL sites). Текущая модель использует placeholder логику. Необходимо интегрировать экспериментальные данные для детерминированного выбора сайтов загрузки.

## Method

1. **Data Integration**
   - Integrate NIPBL ChIP-seq datasets
   - Map ChIP-seq peaks to genomic coordinates
   - Extract sequence features around peaks

2. **TE-Motif Analysis**
   - Compare TE-motif distribution at NIPBL sites vs random
   - Identify enriched motifs
   - Build motif-to-site correlation map

3. **Model Construction**
   - Build Loader Map v1.0 module
   - Implement site selection algorithm
   - Create scoring function for potential sites

## Deliverables

- `archcode_core/loader_map.py` - Main loader map module
- `archcode_core/nipbl_sites.py` - NIPBL site data integration
- `config/archcode_engine.yaml` - Updated with loader parameters
- `tests/test_loader_map.py` - Unit tests

## Constraints

- Использовать модульную архитектуру

- Строгие интерфейсы (dataclass, Enum)
- Production-quality Python code
- Zero-trust: никаких магических констант

## Dependencies

- NIPBL ChIP-seq datasets
- TE annotation data (RepeatMasker/Dfam)
- Genomic coordinate mapping utilities
